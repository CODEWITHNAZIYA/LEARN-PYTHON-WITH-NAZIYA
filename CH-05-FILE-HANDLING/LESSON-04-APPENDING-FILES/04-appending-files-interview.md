# Quick Answers

**Q:** Which mode appends data?

**A:** `"a"`

---

**Q:** Does append mode remove old data?

**A:** No.

---

**Q:** Where is new data added?

**A:** At the end of the file.

---

**Q:** Can append mode create a new file?

**A:** Yes.

---

**Q:** Give one use of append mode.

**A:** Saving logs, notes, attendance records, or feedback.
