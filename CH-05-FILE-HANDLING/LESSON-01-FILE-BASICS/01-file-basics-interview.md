# Quick Answers

**Q:** What is File Handling?

**A:** File handling is the process of creating, reading, writing, and managing files.

---

**Q:** Which function opens a file?

**A:** `open()`

---

**Q:** Which function closes a file?

**A:** `close()`

---

**Q:** What happens if a file doesn't exist in `r` mode?

**A:** Python raises a `FileNotFoundError`.

---

**Q:** Why is file handling important?

**A:** It helps store and retrieve data permanently.
