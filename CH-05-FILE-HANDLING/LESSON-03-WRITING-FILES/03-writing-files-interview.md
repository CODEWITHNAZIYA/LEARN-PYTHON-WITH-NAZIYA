# Quick Answers

**Q:** Which mode is used for writing?

**A:** `"w"`

---

**Q:** Which method writes data into a file?

**A:** `write()`

---

**Q:** Does `"w"` overwrite old data?

**A:** Yes.

---

**Q:** Can `"w"` create a new file?

**A:** Yes.

---

**Q:** Why is writing to files useful?

**A:** It allows programs to save data permanently.
