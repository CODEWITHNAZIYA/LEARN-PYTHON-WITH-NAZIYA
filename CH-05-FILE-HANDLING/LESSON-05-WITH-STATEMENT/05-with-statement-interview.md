# Quick Answers

**Q:** What is the `with` statement?

**A:** It is a statement that automatically closes a file after use.

---

**Q:** Why should we use `with`?

**A:** It makes file handling safer and simpler.

---

**Q:** Do we need `close()` with `with`?

**A:** No.

---

**Q:** Can `with` be used with `"r"`, `"w"` and `"a"` modes?

**A:** Yes.

---

**Q:** Which function is used with `with`?

**A:** `open()`
