# Quick Answers

**Q:** Which mode is used to read files?

**A:** `r`

---

**Q:** Which method reads the complete file?

**A:** `read()`

---

**Q:** Which method reads only one line?

**A:** `readline()`

---

**Q:** Which method returns a list?

**A:** `readlines()`

---

**Q:** Which exception occurs if a file doesn't exist?

**A:** `FileNotFoundError`
