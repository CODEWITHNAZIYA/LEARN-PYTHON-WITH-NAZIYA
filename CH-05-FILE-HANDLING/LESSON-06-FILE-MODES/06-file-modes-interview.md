# Quick Answers

**Q:** Which mode reads a file?

**A:** `r`

---

**Q:** Which mode overwrites old data?

**A:** `w`

---

**Q:** Which mode keeps old data?

**A:** `a`

---

**Q:** Which mode creates a new file only if it doesn't already exist?

**A:** `x`

---

**Q:** Which statement automatically closes files?

**A:** `with`
