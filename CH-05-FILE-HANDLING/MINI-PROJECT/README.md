# 📝 Mini Project - Notes Manager

## 📌 About

**Notes Manager** is a beginner-friendly Python project that allows users to save and manage notes using text files.

This project demonstrates the practical use of **File Handling** in Python.

---

## ✨ Features

- ➕ Add Notes
- 📖 View Notes
- 🗑️ Clear All Notes
- 🚪 Exit Program

---

## 📚 Concepts Used

- File Handling
- Text Files
- Functions
- Loops
- If-Else
- with Statement

---

## ▶️ How to Run

Run the following command:

```bash
python notes-manager.py
```
