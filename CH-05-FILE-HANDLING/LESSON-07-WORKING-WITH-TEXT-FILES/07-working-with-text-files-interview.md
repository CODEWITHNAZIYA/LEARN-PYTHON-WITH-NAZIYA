# Quick Answers

**Q:** What is a text file?

**A:** A text file stores data in plain text format.

---

**Q:** Which mode reads a text file?

**A:** `r`

---

**Q:** Which mode appends data?

**A:** `a`

---

**Q:** Which function counts lines?

**A:** `readlines()`

---

**Q:** Which function helps count words?

**A:** `split()`
