
# 🗳️ Mini Project - Voting Eligibility Checker

## 📌 About

This is a beginner-friendly Python mini project.

The program checks whether a person is eligible to vote based on their:

- Name
- Age
- Country

---

## 📚 Concepts Used

- Variables
- Input & Output
- Type Casting
- If Statement
- Nested If
- Comparison Operators
- String Methods

---

## ▶️ How to Run

Step 1

Open the project folder.

Step 2

Run the program.

```bash
python voting-eligibility-checker.py
```
