
# Lesson 01 - If Statement

## What is an If Statement?

The `if` statement is used to make decisions in Python.

It executes a block of code only when the given condition is **True**.

---

# Syntax

```python
if condition:
    # code
```

---

# Flow

```
Condition

↓

True ?

↓

Yes → Execute Code

↓

No → Skip Code
```

---

# Example 1

```python
age = 20

if age >= 18:
    print("Eligible to Vote")
```

Output

```
Eligible to Vote
```

---

# Example 2

```python
marks = 90

if marks >= 35:
    print("Pass")
```

Output

```
Pass
```

---

# Example 3

```python
number = 10

if number > 0:
    print("Positive Number")
```

---

# Important Points

✔ The condition must return `True` or `False`.

✔ Python uses indentation after `if`.

✔ Indentation is usually **4 spaces**.

---

# Common Mistakes

❌ Missing colon

```python
if age >= 18
```

✔ Correct

```python
if age >= 18:
```

---

❌ Wrong indentation

```python
if age >= 18:
print("Eligible")
```

✔ Correct

```python
if age >= 18:
    print("Eligible")
```

---

# Tips

- Always use meaningful conditions.
- Keep indentation consistent.
- Test different values to understand the output.
