# Quick Answers

**Q:** What is an if statement?

**A:** An if statement is used to execute a block of code only when a condition is True.

---

**Q:** What happens if the condition is False?

**A:** The code inside the if block is skipped.

---

**Q:** Is indentation required in Python?

**A:** Yes. Python uses indentation to define code blocks.

---

**Q:** Which symbol ends an if condition?

**A:** A colon (`:`).

---

**Q:** Can an if statement work without else?

**A:** Yes.
