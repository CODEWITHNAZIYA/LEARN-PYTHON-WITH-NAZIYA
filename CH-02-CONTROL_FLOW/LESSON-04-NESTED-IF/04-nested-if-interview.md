# Quick Answers

**Q:** What is a Nested If?

**A:** A Nested If is an `if` statement inside another `if` statement.

---

**Q:** When does the inner `if` execute?

**A:** Only when the outer `if` condition is True.

---

**Q:** Is indentation required?

**A:** Yes. Proper indentation is mandatory in Python.

---

**Q:** Where is Nested If commonly used?

**A:** Login systems, permission checks, banking, and student management systems.

---

**Q:** Can we write multiple Nested If statements?

**A:** Yes, but avoid excessive nesting to keep code readable.
