# Quick Answers

**Q:** What is `match-case`?

**A:** It is a decision-making statement introduced in Python 3.10.

---

**Q:** What does `case _` represent?

**A:** It is the default case, similar to `else`.

---

**Q:** Which Python version supports `match-case`?

**A:** Python 3.10 and later.

---

**Q:** Can `match-case` check complex conditions?

**A:** No. It is mainly used for matching fixed values.

---

**Q:** How many cases are executed?

**A:** Only the first matching case.
