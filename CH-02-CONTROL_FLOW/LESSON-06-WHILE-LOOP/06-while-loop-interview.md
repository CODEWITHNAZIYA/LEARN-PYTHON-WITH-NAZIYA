# Quick Answers

**Q:** What is a `while` loop?

**A:** A `while` loop repeats a block of code as long as the condition is True.

---

**Q:** What causes an infinite loop?

**A:** When the loop condition never becomes False.

---

**Q:** When should you use a `while` loop?

**A:** When the number of iterations is unknown.

---

**Q:** Can a `while` loop execute zero times?

**A:** Yes. If the condition is False at the beginning, the loop body will not run.

---

**Q:** Is indentation mandatory?

**A:** Yes. Python uses indentation to define the loop body.
