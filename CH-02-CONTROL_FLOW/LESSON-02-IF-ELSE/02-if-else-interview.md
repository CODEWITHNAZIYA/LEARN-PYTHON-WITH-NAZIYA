# Quick Answers

**Q:** What is an if-else statement?

**A:** It is used to choose between two possible outcomes based on a condition.

---

**Q:** When is the `else` block executed?

**A:** When the condition in the `if` statement is False.

---

**Q:** Is indentation required?

**A:** Yes. Python uses indentation to define code blocks.

---

**Q:** Can only one block run?

**A:** Yes. Either the `if` block or the `else` block runs.

---

**Q:** Can an `else` statement be used without an `if` statement?

**A:** No.
