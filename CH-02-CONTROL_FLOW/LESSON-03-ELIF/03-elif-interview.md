# Quick Answers

**Q:** What is `elif`?

**A:** `elif` stands for **else if** and is used to check multiple conditions.

---

**Q:** Can we use multiple `elif` blocks?

**A:** Yes.

---

**Q:** Is `else` required?

**A:** No, it is optional.

---

**Q:** How many blocks execute?

**A:** Only one matching block is executed.

---

**Q:** Can `elif` be used without `if`?

**A:** No.
