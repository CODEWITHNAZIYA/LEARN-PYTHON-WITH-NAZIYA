# Quick Answers

**Q:** What is `break`?

**A:** `break` is used to immediately exit a loop.

---

**Q:** Which loops support `break`?

**A:** Both `for` and `while` loops.

---

**Q:** Can `break` be used outside a loop?

**A:** No. It will raise an error.

---

**Q:** Does `break` stop the program?

**A:** No. It only exits the current loop.

---

**Q:** What is the difference between `break` and `continue`?

**A:** `break` exits the loop completely, while `continue` skips the current iteration and moves to the next one.
