# Quick Answers

**Q:** What is a `for` loop?

**A:** A `for` loop repeats a block of code for each item in a sequence.

---

**Q:** Which function is commonly used with a `for` loop?

**A:** `range()`

---

**Q:** Can a `for` loop iterate through a string?

**A:** Yes.

---

**Q:** What is the default start value of `range(5)`?

**A:** `0`

---

**Q:** Is indentation mandatory?

**A:** Yes. Python uses indentation to define the loop body.
