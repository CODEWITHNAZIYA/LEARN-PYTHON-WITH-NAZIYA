# Quick Answers

**Q:** What is `pass`?

**A:** `pass` is a placeholder statement that does nothing.

---

**Q:** Does `pass` execute any code?

**A:** No.

---

**Q:** Can `pass` be used in functions and classes?

**A:** Yes.

---

**Q:** Does `pass` stop a loop?

**A:** No. It simply does nothing and execution continues.

---

**Q:** Why is `pass` useful?

**A:** It allows you to write code structures first and implement them later without causing syntax errors.
