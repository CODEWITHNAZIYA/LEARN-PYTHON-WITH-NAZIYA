# Quick Answers

**Q:** What is `return`?

**A:** It sends a value back from a function.

---

**Q:** Does `return` end the function?

**A:** Yes. The function stops immediately after `return`.

---

**Q:** What is the difference between `print()` and `return`?

**A:** `print()` displays output, while `return` sends a value back that can be reused.

---

**Q:** Can a function return different data types?

**A:** Yes. It can return strings, numbers, lists, dictionaries, Boolean values, and more.

---

**Q:** Is `return` required in every function?

**A:** No. A function can work without `return`, but it won't send a value back.
