# Quick Answers

**Q:** What are keyword arguments?

**A:** Arguments passed using parameter names.

---

**Q:** Does the order matter?

**A:** No. The order does not matter when using keyword arguments.

---

**Q:** Can positional and keyword arguments be used together?

**A:** Yes.

---

**Q:** Why are keyword arguments useful?

**A:** They improve readability and reduce mistakes.

---

**Q:** What happens if the same parameter is passed twice?

**A:** Python raises a `TypeError`.
