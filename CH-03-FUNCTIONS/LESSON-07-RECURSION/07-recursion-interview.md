# Quick Answers

**Q:** What is recursion?

**A:** Recursion is a technique where a function calls itself until a base case is reached.

---

**Q:** What is a base case?

**A:** It is the stopping condition of a recursive function.

---

**Q:** What happens without a base case?

**A:** Python keeps calling the function until a `RecursionError` occurs.

---

**Q:** Can recursion replace loops?

**A:** Sometimes, but loops are often simpler for basic repetition.

---

**Q:** Where is recursion commonly used?

**A:** Factorial, Fibonacci, trees, graph traversal, and divide-and-conquer algorithms.
