# Quick Answers

**Q:** What are default arguments?

**A:** They are parameters with predefined values.

---

**Q:** Can default values be overridden?

**A:** Yes. Pass a new value while calling the function.

---

**Q:** What happens if no value is passed?

**A:** Python uses the default value.

---

**Q:** Where should default parameters be placed?

**A:** After all required parameters.

---

**Q:** Why are default arguments useful?

**A:** They reduce repetitive code and make functions easier to use.
