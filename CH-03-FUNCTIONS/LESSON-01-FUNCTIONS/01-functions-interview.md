# Quick Answers

**Q:** What is a function?

**A:** A function is a reusable block of code that performs a specific task.

---

**Q:** Which keyword is used to create a function?

**A:** `def`

---

**Q:** Does a function run automatically?

**A:** No. It runs only when it is called.

---

**Q:** Can a function be called multiple times?

**A:** Yes.

---

**Q:** Why are functions important?

**A:** They reduce code repetition, improve readability, and make programs easier to maintain.
