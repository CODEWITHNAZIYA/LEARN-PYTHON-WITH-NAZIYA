# Quick Answers

**Q:** What is a lambda function?

**A:** A lambda function is a small anonymous function written in a single line.

---

**Q:** Why is it called anonymous?

**A:** Because it has no function name.

---

**Q:** Can a lambda function have multiple arguments?

**A:** Yes.

---

**Q:** Does a lambda function need `return`?

**A:** No. It automatically returns the result.

---

**Q:** Where are lambda functions commonly used?

**A:** With `map()`, `filter()`, and `sorted()`.
