# 🧮 Mini Project - Calculator Using Functions

## 📌 About

This is a beginner-friendly calculator project built using Python functions.

The program performs four basic mathematical operations:

- Addition
- Subtraction
- Multiplication
- Division

---

## 📚 Concepts Used

- Functions
- Parameters
- Return Statement
- If-Elif-Else
- User Input
- Type Casting

---

## ▶️ How to Run

Open the project folder.

Run the following command:

```bash
python calculator-using-functions.py
```
