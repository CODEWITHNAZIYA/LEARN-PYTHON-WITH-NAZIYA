# Quick Answers

**Q:** What is a parameter?

**A:** A parameter is a variable defined inside a function to receive values.

---

**Q:** What is an argument?

**A:** An argument is the actual value passed to a function.

---

**Q:** Can a function have multiple parameters?

**A:** Yes.

---

**Q:** What happens if you don't pass the required argument?

**A:** Python raises a `TypeError`.

---

**Q:** Why are parameters useful?

**A:** They make functions reusable, flexible, and easier to maintain.
