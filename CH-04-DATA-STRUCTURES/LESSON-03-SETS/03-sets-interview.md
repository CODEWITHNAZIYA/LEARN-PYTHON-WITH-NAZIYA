# Quick Answers

**Q:** What is a set?

**A:** A set is an unordered collection of unique items.

---

**Q:** Are duplicate values allowed?

**A:** No.

---

**Q:** Which brackets are used for sets?

**A:** Curly braces `{}`.

---

**Q:** What does `union()` do?

**A:** It combines all unique elements from two sets.

---

**Q:** When should sets be used?

**A:** When you need unique values or fast membership checking.
