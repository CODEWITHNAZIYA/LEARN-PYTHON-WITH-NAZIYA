# Quick Answers

**Q:** What is a tuple?

**A:** A tuple is an ordered and immutable collection of items.

---

**Q:** Can tuple values be changed?

**A:** No.

---

**Q:** Which brackets are used for tuples?

**A:** Parentheses `()`.

---

**Q:** Name two tuple methods.

**A:** `count()` and `index()`.

---

**Q:** When should tuples be used?

**A:** When the data should not change.
