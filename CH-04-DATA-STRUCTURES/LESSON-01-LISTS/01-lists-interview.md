# Quick Answers

**Q:** What is a list?

**A:** A list is an ordered and mutable collection of items.

---

**Q:** Are duplicate values allowed?

**A:** Yes.

---

**Q:** Can a list store different data types?

**A:** Yes.

---

**Q:** Which method adds an item to the end of a list?

**A:** `append()`

---

**Q:** Which function returns the length of a list?

**A:** `len()`
