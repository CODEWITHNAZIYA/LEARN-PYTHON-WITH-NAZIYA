# Quick Answers

**Q:** What is a dictionary?

**A:** A dictionary is a mutable collection of key-value pairs.

---

**Q:** Are duplicate keys allowed?

**A:** No. Keys must be unique.

---

**Q:** Which brackets are used?

**A:** Curly braces `{}`.

---

**Q:** Which method returns all keys?

**A:** `keys()`.

---

**Q:** Why are dictionaries useful?

**A:** They make data easy to organize and retrieve using keys.
