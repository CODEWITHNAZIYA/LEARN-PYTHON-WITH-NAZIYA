# 🎓 Mini Project - Student Management System

## 📌 About

This is a beginner-friendly **Student Management System** built using Python.

The project allows users to manage student records through a simple menu-driven program.

---

## ✨ Features

- ➕ Add Student
- 📋 View All Students
- 🔍 Search Student
- ❌ Delete Student
- 🚪 Exit Program

---

## 📚 Concepts Used

- Lists
- Dictionaries
- Functions
- Loops
- If-Else Statements
- User Input

---

## ▶️ How to Run

Open the project folder and run:

```bash
python student-management-system.py
```
