
# Lesson 01 - Variables

## What is a Variable?

A variable is a container used to store data in a program.

Think of it like a labeled box where you can keep information.

Example

```python
name = "Naziya"
age = 21
```

---

# Why Do We Use Variables?

Variables help us store and reuse data.

Instead of writing the same value again and again, we can store it in a variable.

---

# Rules for Naming Variables

✔ Variable names can contain:

- Letters
- Numbers
- Underscore (_)

✔ Variable names cannot:

- Start with a number
- Contain spaces
- Use Python keywords

Example

```python
name = "Naziya"

student_age = 21

marks1 = 90
```

Wrong

```python
1name = "Python"

student age = 20

if = 10
```

---

# Creating Variables


# Changing Variable Value

```python
course = "Python"

course = "Python Programming"

print(course)
```

Output

```
Python Programming
```

---

# Multiple Variables

```python
x, y, z = 10, 20, 30
```

---

# Checking Variable Type

```python
name = "Python"

print(type(name))
```

Output

```
<class 'str'>
```

# Common Mistakes

❌ Starting a variable name with a number

```python
2name = "Naziya"
```

❌ Using spaces

```python
student name = "Ali"
```

❌ Using keywords

```python
if = 10
```

---

# Tips

- Use meaningful variable names.
- Keep names short and readable.
- Follow snake_case style.

Example

```python
student_name = "Naziya"
```
