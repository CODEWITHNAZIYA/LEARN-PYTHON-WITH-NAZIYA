# Quick Answers

**Q:** What is a variable?

**A:** A variable is a container used to store data.

---

**Q:** Which function checks the data type?

**A:** `type()`

---

**Q:** Can variable values be changed?

**A:** Yes.

---

**Q:** Which naming style is recommended?

**A:** `snake_case`

---

**Q:** Can variables store different types of data?

**A:** Yes, such as strings, integers, floats, and booleans.
