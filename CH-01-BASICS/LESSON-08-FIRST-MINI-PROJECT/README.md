
# 🎓 Mini Project - Student Information System

## 📖 About This Project

This is my first Python mini project created after learning the basics of Python.

The project takes student information as input, calculates the total marks and percentage, and displays the final result.

---

# 📚 Concepts Used

- Variables
- Input & Output
- Type Casting
- Arithmetic Operators
- Strings
- Comments
- if-else Condition
- f-Strings

---

# 🚀 Features

- Accepts student details
- Takes marks of three subjects
- Calculates total marks
- Calculates percentage
- Displays Pass/Fail result
- Beginner-friendly code
