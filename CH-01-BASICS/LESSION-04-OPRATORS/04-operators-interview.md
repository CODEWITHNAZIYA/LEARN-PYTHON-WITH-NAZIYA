## Quick Answers

**Q:** What is an operator?

**A:** An operator is a symbol used to perform operations on variables and values.

---

**Q:** Which operator is used for addition?

**A:** `+`

---

**Q:** Which operator is used for comparison?

**A:** `==`

---

**Q:** Which operator checks object identity?

**A:** `is`

---

**Q:** Which operator checks membership?

**A:** `in`
