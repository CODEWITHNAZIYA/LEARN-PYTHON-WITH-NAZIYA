# Quick Answers

**Q:** What is a string?

**A:** A string is a sequence of characters used to store text.

---

**Q:** Which function returns the length of a string?

**A:** `len()`

---

**Q:** Which method converts text to uppercase?

**A:** `upper()`

---

**Q:** Which method replaces a word?

**A:** `replace()`

---

**Q:** Are Python strings mutable?

**A:** No, strings are immutable.
