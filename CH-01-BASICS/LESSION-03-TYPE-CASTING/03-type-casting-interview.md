# Quick Answers

**Q:** What is type casting?

**A:** Type casting is the process of converting one data type into another.

---

**Q:** Which function converts to integer?

**A:** `int()`

---

**Q:** Which function converts to float?

**A:** `float()`

---

**Q:** Which function converts to string?

**A:** `str()`

---

**Q:** Why is type casting important?

**A:** It helps us work with the correct data type, especially when taking user input.
