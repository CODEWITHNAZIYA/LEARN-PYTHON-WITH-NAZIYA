# Quick Answers

**Q:** Which function displays output?

**A:** `print()`

---

**Q:** Which function takes input?

**A:** `input()`

---

**Q:** What does input() return by default?

**A:** String (`str`)

---

**Q:** Which function converts string to integer?

**A:** `int()`

---

**Q:** Which function converts string to float?

**A:** `float()`
