# Quick Answers

**Q:** What is a keyword?

**A:** A keyword is a reserved word that has a special meaning in Python.

---

**Q:** Can keywords be used as variable names?

**A:** No.

---

**Q:** Which module shows all Python keywords?

**A:** `keyword`

---

**Q:** Which keyword is used to create a function?

**A:** `def`

---

**Q:** Which keyword returns a value from a function?

**A:** `return`
