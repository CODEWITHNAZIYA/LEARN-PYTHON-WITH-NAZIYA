# Quick Answers

**Q:** What is a comment?

**A:** A comment is a note that explains the code. Python ignores comments during execution.

---

**Q:** Which symbol starts a single-line comment?

**A:** `#`

---

**Q:** Are comments executed by Python?

**A:** No.

---

**Q:** Why are comments important?

**A:** They make code easier to read, understand, and maintain.

---

**Q:** Can comments help in teamwork?

**A:** Yes. They help other developers understand the code quickly.
